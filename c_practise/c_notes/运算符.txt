     算术运算符
     +  -   *  /   %

     关系运算符
      >     >=   <     <=     !=(不等于）   ==（等于）
     
     逻辑运算符
      ！（非）    &&（并且，即与）  ||（或）
      ！真            假 
      ！假            真
      
      C语言对真假的处理
         非零是真
         零是假

         真用1表示
         假用0表示
         &&左边的表达式为假时 右边的表达式不执行
         ||左边的表达式为真时 右边的表达式不执行
     赋值运算符
        =  +=  *=  /=  -=  


     优先级别：
         算术〉关系〉逻辑〉赋值
